# MSB Help — AI-ассистент для малого бизнеса Казахстана

Веб-приложение, которое помогает владельцам малого бизнеса в Казахстане принимать решения по маркетингу, финансам и развитию через диалог с искусственным интеллектом и анализ данных.

🌐 **Демо:** [dilnaz-project.vercel.app](https://dilnaz-project.vercel.app)

---

## Зачем это нужно

Малому бизнесу в Казахстане часто не хватает доступа к консультантам по маркетингу и финансам — они дороги или находятся в крупных городах. Платформа закрывает эту потребность: даёт круглосуточный AI-консультант, который понимает местные реалии (Kaspi, 2ГИС, Даму, тенге) и говорит на трёх языках.

**Целевая аудитория:** предприниматели 40+ с низкой цифровой грамотностью. Под них спроектирован весь UX — простой интерфейс, ассистент сначала задаёт вопросы по одному, потом даёт готовое решение без шаблонных заглушек.

---

## Что умеет

**Чат с AI-консультантом.** Три категории: маркетинг, финансы, бизнес-идеи. Под каждую — отдельный системный промпт, заточенный под казахстанский контекст. Ответы стримятся в реальном времени (печатаются по буквам). Ассистент сам ведёт диалог: расспрашивает о бизнесе, затем предлагает персональное решение.

**Анализ бизнеса по Excel.** Пользователь загружает таблицу с данными, заполняет короткую анкету (бизнес, аудитория, проблема, цель) — AI возвращает структурированный анализ: ключевые метрики, графики динамики и прогноза, выявленные проблемы и конкретные рекомендации.

**Гостевой чат.** Можно попробовать продукт без регистрации.

**Личный кабинет.** Регистрация, история чатов, смена пароля, удаление аккаунта.

**Мультиязычность.** Русский, английский, казахский. Переключение в один клик.

**Тёмная и светлая темы.**

## Безопасность и защита данных

Это одна из ключевых задач проекта, особенно с учётом аудитории.

- **Хеширование паролей** через bcrypt — даже при утечке базы пароли нечитаемы.
- **JWT-токены** для авторизации с истечением 7 дней.
- **Анонимизация Excel перед AI.** Перед отправкой данных в Claude API из таблиц автоматически удаляются персональные данные клиентов: телефоны, email, ИИН. Реальные ПДн пользователей не покидают сервер.
- **Файлы — только в память.** Загруженные Excel не сохраняются на диск, обрабатываются в оперативной памяти и сразу очищаются.
- **Ограничения.** Лимит на размер файла (10 МБ), типы файлов (только Excel/CSV), частоту запросов (rate limiting), длину сессии.
- **Helmet** для защиты от частых веб-атак (XSS, clickjacking).
- **CORS по белому списку** — сервер принимает запросы только с разрешённых адресов.
- **Валидация входных данных** — на сервере проверяется формат email, длина и сложность пароля, длина имени.
- **Защищённые маршруты** — доступ к чатам, анализу и профилю требует авторизации.

## Технологии

**Frontend:** React 19, Vite, Tailwind CSS 4, React Router, GSAP (анимации), Chart.js (графики), Lucide (иконки), jsPDF (экспорт анализа).

**Backend:** Node.js, Express 5, MongoDB через Mongoose, JWT, bcrypt, Multer (загрузка файлов), XLSX (парсинг таблиц), Helmet, express-rate-limit.

**AI:** Anthropic Claude API (стриминг).

**Хостинг:** Vercel (frontend + serverless backend), MongoDB Atlas (база данных).

## Запуск локально

**Требования:** Node.js 18+, доступ к MongoDB (локально или Atlas), API-ключ Anthropic.

**1. Клонировать репозиторий**

```bash
git clone https://github.com/DilnazU/dilnaz-project.git
cd dilnaz-project
```

**2. Настроить переменные окружения**

Создайте файл `server/.env` со следующим содержимым:

```
MONGODB_URI=mongodb+srv://<логин>:<пароль>@<кластер>/<база>
JWT_SECRET=<любая длинная случайная строка>
ANTHROPIC_API_KEY=<ваш ключ от Anthropic>
CLIENT_URL=http://localhost:5173
PORT=5000
```

**3. Установить зависимости и запустить**

В двух терминалах:

```bash
# Терминал 1 — сервер
cd server
npm install
npm run dev
```

```bash
# Терминал 2 — клиент
cd client
npm install
npm run dev
```

Открыть в браузере: `http://localhost:5173`

## API

Основные эндпоинты (полный список — в `server/routes/`):

| Метод | Путь | Что делает |
|-------|------|------------|
| `POST` | `/auth/signup` | Регистрация |
| `POST` | `/auth/login` | Вход |
| `GET` | `/auth/me` | Данные текущего пользователя |
| `PUT` | `/auth/password` | Смена пароля |
| `DELETE` | `/auth/account` | Удаление аккаунта |
| `POST` | `/api/chat` | Сообщение в чат (стриминг) |
| `GET` | `/api/chat` | Список чатов пользователя |
| `POST` | `/api/chat/guest` | Гостевой чат |
| `POST` | `/api/analysis` | Анализ Excel-файла |
| `GET` | `/health` | Проверка состояния сервера |
